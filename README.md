# News AI Summarizer

This project is an AI-powered news summarizer. 

## Setup and Running

1. Install the required dependencies.
2. Run `app.py` or `run_ai.py` to test the summarization.
3. Use `train_summarizer.py` to train the model on your dataset.

## Large Files (Ignored in Git)

Due to GitHub's file size limits, the datasets and trained model checkpoints are not included in this repository. 
You will need to re-download the datasets (like `news_summary.csv` and `news_summary_more.csv`) and run the training scripts to generate the `my_news_summarizer/` model folder locally. Alternatively, you can host these large files on a platform like Hugging Face Hub.
